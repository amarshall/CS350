Lab 5: Modules and /proc entries

Description

Write a kernel module that creates a /proc entry /proc/hello. When a user program tries to read /proc/hello, it receives the string "Hello World". Test whether your module works using the command 

$ cat  /proc/hello

Hints

  • Basics of writing a module can be found in lecture slides.
  • Examples of simple kernel modules can be found here: http://www.cs.binghamton.edu/~kartik/cs350/examples/module/
  • Example of how to implement a function, which responds to a read operation on a /proc entry, is in the fortune_read function here: http://www.ibm.com/developerworks/linux/library/l-proc.html . Pay attention to the sprintf(...) statement and the return statement.
  • Using create_proc_read_entry instead of create_proc_entry can simplify your coding for this lab.
  • If you get stuck, ask TAs for help.
