Lab 6 - Catch signals

Write a small program that:

  • Busy waits on while loop
  • When user press Ctrl-C or Ctrl-Z, it prints out "OH NO!!!" and continue the busy wait
Hint: write signal handlers for SIGINT (Ctrl-C) and SIGTSTP (Ctrl-Z)
For information on signal, refer to:
http://www.cs.cf.ac.uk/Dave/C/node24.html
To terminate the program, type "kill -9 [YOUR_PROCESS_ID]". This command sends SIGKILL signal which cannot be caught by your program!
